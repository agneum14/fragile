Hilfeseite

1. Funktionalität der Kopfzeilenleiste

Datei Dropdown-Menü:

Wenn Sie den Kopfbereich von Fragile betrachten, sehen Sie drei verschiedene Dropdown-Menüs zur Auswahl. Das erste davon ist das Dropdown-Menü Datei. Wenn Sie über das Dropdown-Menü Datei scrollen, wird Ihnen ein Beenden-Button angezeigt. Mit diesem Button wird das Programm geschlossen. Eine alternative Möglichkeit, Fragile zu schließen, besteht darin, einfach das Fenster oben rechts im Rahmen zu schließen.

Ansicht Dropdown-Menü:

Wenn Sie über das Dropdown-Menü Ansicht schweben, sehen Sie den Kreisdiagramm-Button. Dieser Button wird es Ihnen ermöglichen, das Ergebnis Ihrer gemischten Bruchberechnung in Form eines Kreisdiagramms zu betrachten. Es zeigt Ihre Ganzzahlen als volle Kreise und Ihren potenziell verbleibenden Bruch als Prozentsatz des Kreisdiagramms.

2. Layout des Taschenrechners für gemischte Brüche

Button-Layout:

Hier haben wir die Hauptoberfläche des Taschenrechners für gemischte Brüche. Wie die meisten Taschenrechner verfügt er über eine Tastatur mit Softbuttons, mathematischen Operationen und einer "="-Taste. Dieser Taschenrechner verfügt auch über eine zusätzliche Taste unten in der Reihe. Dies ist die Bruch / Fokus ändern-Taste.

Bruch-Taste:

Da dies ein Taschenrechner für gemischte Brüche ist, benötigen wir eine Möglichkeit, diese Brüche zu erstellen. Der Weg, auf dem Fragile dies erreicht, besteht darin, dem Benutzer zu ermöglichen, den Fokus darauf zu legen, welche Zahl im Bruch sie ändern möchten. Wie durch das Rechteck im Anzeigerahmen unten gezeigt.

Fokusänderung:

Durch Drücken der Bruch-Taste wird das Rechteck zum Zähler, dann zum Nenner bewegt und dann zurück zur Position des Ganzzahls. Dies ermöglicht es dem Benutzer, eine beliebige Zahl beliebig oft zu ändern, bevor er eine Berechnung durchführt.

3. Durchführung einer Berechnung

Erste Schritte:

Um mit der Durchführung einer Berechnung zu beginnen, müssen Sie die erste gemischte Zahl eingeben, auf der Sie die Berechnung durchführen möchten. Füllen Sie dazu den Zähler und den Nenner mithilfe der zuvor genannten Fokus-Funktion aus.

Gültiges Eingabebeispiel / Fehlermeldung:

Dies ist ein Beispiel für eine gültige Eingabe. Wenn eine ungültige Eingabe erfolgt, wird der Benutzer mit dieser Fehlermeldung begrüßt.

Mathematische Operationen:

Nachdem eine gültige Eingabe erfolgt ist, wählen Sie die auszuführende mathematische Operation aus.

Das Ergebnis:

Nachdem die mathematische Operation ausgewählt wurde, geben Sie die zweite gemischte Zahl ein, auf der die Berechnung durchgeführt werden soll. Wiederholen Sie den ersten Schritt dieses Prozesses. Nachdem die zweite Bruchzahl eingegeben wurde, drücken Sie die "="-Taste, um den Ausdruck zu bewerten. Das resultierende Ergebnis wird wie folgt dargestellt.
